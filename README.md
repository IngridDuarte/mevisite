# mevisite

## Como instalar
```
$ cd meviste
$ npm install
```

## Dependencias globais necessárias
```
$ npm install -g httpserver jsonserver
```

## Rodando o site do projeto
```
$ npm run start:site
```

## Rodando a API
```
$ npm run start:api
```

## Padrão de mensagem de commit
```
"[@githubuser] - messagem de commit"
```
